Koala::Utility::File::IsValid
=============================

.. csv-table::
	
	"**bool IsValid() const;**"

Check if the file is valid.

Return Value
------------

If the object has valid file, it returns :code:`true`. If the file creation is failed or the file is deleted, it returns :code:`false`.
